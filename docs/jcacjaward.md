# 学会賞
2007年より毎年１回、地図学の理論構築と普及、地図・空間情報などに関する技術の向上に尽力された個人や法人、団体を日本地図学会として表彰しています。論文賞、論文奨励賞、野村正七地図賞、作品・出版賞、教育普及賞、功労賞、特別賞といった多様な評価軸で、受賞理由も含めてすべてこちらにアーカイブしております。
